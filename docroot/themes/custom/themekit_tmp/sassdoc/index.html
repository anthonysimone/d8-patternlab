    variables

    base-font-size

    $base-font-size: 16px;

    Description

    The font size that all elements will inherit. This is NOT the same as the font size that is set on the html element.

    base-line-height

    $base-line-height: 1.5;

    Description

    This line-height determines the basic unit of vertical rhythm. Line heights should remain unitless.

    Links

    • https://developer.mozilla.org/en-US/docs/Web/CSS/line-height#Prefer_unitless_numbers_for_line-height_values

    key

    @function key($map, $key, $sub-key: null) { ... }

    Description

    Return a map from the specified key. Optionally specify a sub key. Works similarly to map-get but with support for sub keys.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $map—noneMap—none
    $key

    The map key to retrieve.

    		String—none
    $sub-key

    A sub-key of $key

    		Stringnull

    Returns

    * or Null —

    The value of the supplied key

    bp

    @mixin bp($size, $default: false, $direction: 'min-width') { ... }

    Description

    Create common breakpoints and optionally set a fallback for non-supporting browsers.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $size

    A key in the $breakpoints variable or any valid css unit.

    		String—none
    $default

    Determines if this breakpoint should be used for a no-query fallback.

    		Booleanfalse
    $direction

    Determines the direction of the breakpoint. Can be min-width or max-width.

    		String'min-width'

    Content

    This mixin allows extra content to be passed (through the @content directive).
